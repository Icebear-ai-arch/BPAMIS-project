## 📌 Overview
The Barangay Panducot Adjudication Management Information System is a capstone project developed to improve the management of barangay-level complaints, hearings, and case records. It aims to digitalize the traditional manual process, making it faster, more organized, and more efficient.

---

## 🚀 Features
- Resident and External Complainant Management  
- Complaint Filing and Tracking  
- Case Management System  
- Hearing Scheduling and Logs  
- User Roles (Admin, Secretary, Officials)  
- Notifications System  
- Reports Generation  
- Document and Record Management  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL  
- Tailwind
- XAMPP / Web Hosting  

---

## 🎯 Project Purpose
This system was developed to:
- Digitize barangay complaint and case handling
- Handles the Barangay Justice System
- Improve record organization and retrieval
- Speed up adjudication processes
- Provide a centralized management system for officials
